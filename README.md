# Bonfire:Search

An extension for [Bonfire](https://bonfire.cafe/) to handle:

- Search indexing
- Ultra-fast search
- LiveView search UI

You can implement adapters for your prefered search backend. Adapters are included for:
- [Meili Search](https://www.meilisearch.com/)
